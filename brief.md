## 1. ЦЕЛЬ САЙТА И ЦЕЛЕВАЯ АУДИТОРИЯ

### Цель
Личный сайт-портфолио, который одновременно:
- Строит личный бренд Виктории Полтавченко как эксперта в AI и цифровых технологиях
- Привлекает B2B-клиентов и генерирует входящие заявки
- Демонстрирует компетентность через сам факт качественного сайта

**Главное целевое действие:** написать / забронировать звонок

### Целевая аудитория
**Основная:** Владельцы малого и среднего бизнеса, руководители, маркетологи

**Их задачи:**
- Заказать сайт без технических головных болей
- Автоматизировать рутинные процессы с помощью AI
- Найти одного человека вместо нескольких подрядчиков

**Их страхи (нужно закрыть на странице):**
- «Не разбираюсь в технологиях — вдруг обманут»
- «Непонятно сколько стоит и сколько занимает»
- «Хочу результат, а не процесс»
- «Буду работать вслепую — что будет после того как напишу?»

**Психологическая логика пути:** ВНИМАНИЕ → ИНТЕРЕС → ДОВЕРИЕ → ЖЕЛАНИЕ → ДЕЙСТВИЕ

## 4. СТИЛЬ ДИЗАЙНА

### Направление: «Dark Futurist» с элементами «Clean Architect»

**Обоснование:**
- B2B аудитория → профессиональное, не игривое
- AI/tech услуги → тёмная тема усиливает tech-credibility
- Акцент на доверии → чистота и пространство важнее эффектов

### Цветовая палитра

```
Основные:
  Фон:              #0F172A  (Deep navy-black)
  Поверхность:      #1E293B  (Dark slate — карточки, блоки)
  Граница:          #334155  (Slate border)

Акценты:
  Accent primary:   #6366F1  (Indigo — кнопки, ссылки, hover)
  Accent secondary: #06B6D4  (Cyan — иконки, подчёркивания, теги)

Типографика:
  Текст основной:   #F1F5F9  (Cool white)
  Текст muted:      #94A3B8  (Slate — второстепенный текст)

CSS переменные:
  --bg: #0F172A
  --surface: #1E293B
  --border: #334155
  --accent: #6366F1
  --cyan: #06B6D4
  --text: #F1F5F9
  --muted: #94A3B8
```

### Типографика

```
Шрифт заголовков: Syne (700-800 weight)
  — Широкий, уверенный, design-aware developer
  — Google Fonts, бесплатно

Шрифт текста: DM Sans (400-500 weight)
  — Чистый, читаемый, современный
  — Google Fonts, бесплатно

Шрифт кода/меток: Syne Mono или JetBrains Mono
  — Для технических элементов

Иерархия размеров:
  H1 Hero:    72px / 4.5rem · weight 800 · line-height 1.05 · tracking -0.03em
  H1 Обычный: 56px / 3.5rem · weight 700 · line-height 1.1
  H2 Секция:  40px / 2.5rem · weight 700 · line-height 1.15
  H3 Подзаг:  28px / 1.75rem · weight 600 · line-height 1.25
  Body large: 18px · weight 400 · line-height 1.7
  Body:       16px · weight 400 · line-height 1.65
  Label/Cap:  13px · weight 500 · uppercase · tracking 0.08em

Mobile: H1 → 40-48px, H2 → 28px, body остаётся 16px
```

### Настроение и атмосфера
- Профессиональный, современный, технологичный
- Глубина и пространство — не перегруженность
- Точность и чистота — «качество в деталях»
- Тёплый через человеческий контент, холодный через визуал

## 5. ОБЯЗАТЕЛЬНЫЕ ЭЛЕМЕНТЫ

### Навигация
- [ ] Sticky navbar с glass-эффектом при скролле
- [ ] Логотип-инициалы «ВП» с градиентом
- [ ] CTA-кнопка «Обсудить проект» всегда видна
- [ ] Бургер-меню на мобайле
- [ ] Плавный скролл к секциям (anchor links)

### Кнопки
- [ ] Primary: bg-accent + glow-shadow при hover
- [ ] Secondary/Ghost: border-accent + fill при hover
- [ ] Hover: лёгкое свечение (box-shadow: 0 0 24px rgba(99,102,241,0.5))
- [ ] Active state: scale(0.97)

### Карточки
- [ ] Glassmorphism: backdrop-filter: blur(16px) + border 1px accent/15
- [ ] Hover: translateY(-4px) + border более яркий
- [ ] Иконки с акцентным фоном (bg-accent/10 или bg-cyan/10)

### Анимации (MVP — Phase 1)
- [ ] Fade-up при скролле (IntersectionObserver, нативный, без библиотек)
- [ ] Stagger delay между карточками (+150ms)
- [ ] Hero: мгновенное появление с delay 100ms
- [ ] Aurora gradient mesh на hero (CSS radial-gradient + keyframes)
- [ ] Hover glow на кнопках

### Анимации (V2 — Phase 2)
- [ ] Lenis smooth scroll
- [ ] GSAP SplitText на H1 hero
- [ ] Spotlight hover эффект внутри карточек
- [ ] ScrollTrigger stagger на секциях
- [ ] Счётчики (0 → число) при появлении блока статистики

### Анимации (V3 — Premium)
- [ ] Кастомный курсор (dot + ring + mix-blend-mode difference)
- [ ] Магнитные CTA-кнопки
- [ ] AI-виджет «Задай вопрос моему ИИ-ассистенту»
- [ ] Social Proof Mosaic (marquee в два ряда)

### Форма контакта
- [ ] Максимум 4 поля (имя, контакт, тип, описание)
- [ ] Select для типа проекта
- [ ] Состояния: idle / sending / success / error
- [ ] Подключить Formspree или Resend

### Адаптивность
- [ ] Mobile-first подход
- [ ] Breakpoints: 640px (sm), 768px (md), 1024px (lg), 1280px (xl)
- [ ] Проверить на реальном телефоне
- [ ] Горизонтальный скролл исключить полностью
- [ ] Touch targets минимум 44×44px

### Производительность
- [ ] PageSpeed Insights > 90 (mobile)
- [ ] Шрифты через next/font (оптимизация)
- [ ] Изображения через next/image (lazy loading, WebP)
- [ ] Время загрузки < 3 секунд на мобайле

### SEO и мета
