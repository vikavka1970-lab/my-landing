# DEPLOYMENT.md — Деплой бота на Beget VPS

Пошаговая инструкция. Читай и делай по порядку.

---

## Что куда деплоится

| Компонент | Платформа | Статус |
|---|---|---|
| Лендинг `index.html` | GitHub Pages | ✅ Готово |
| Mini App `tg-app/index.html` | Vercel | ✅ Готово |
| Telegram-бот `bot/bot.js` | Beget VPS | ⬜ Делаем сейчас |

---

## Шаг 1 — Создать VPS на Beget

1. Войти на `cp.beget.com`
2. Облако → Виртуальные серверы → **Создать сервис**
3. Выбрать минимальную конфигурацию:
   - ОС: **Ubuntu 22.04**
   - RAM: 1 GB
   - CPU: 1 ядро
4. Пополнить баланс и оплатить
5. Дождаться письма от Beget с данными:
   - IP-адрес сервера
   - Пароль root

---

## Шаг 2 — Подключиться к серверу

Открыть терминал (или PowerShell на Windows) и выполнить:

```bash
ssh root@ВАШ_IP
```

Ввести пароль из письма Beget.

---

## Шаг 3 — Установить Node.js

```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt-get install -y nodejs
node --version
```

---

## Шаг 4 — Установить PM2

PM2 — менеджер процессов. Он следит за ботом и перезапускает его если что-то пошло не так.

```bash
npm install -g pm2
```

---

## Шаг 5 — Загрузить код бота на сервер

На своём компьютере (не на сервере):

```bash
scp -r bot/ root@ВАШ_IP:/root/brief-bot
```

---

## Шаг 6 — Создать .env на сервере

На сервере:

```bash
cd /root/brief-bot
nano .env
```

Вставить содержимое (значения взять из локального `.env`):

```
BOT_TOKEN=<токен от BotFather>
VICTORIA_CHAT_ID=931351685
```

Сохранить: `Ctrl+O` → Enter → `Ctrl+X`.

---

## Шаг 7 — Установить зависимости и запустить

```bash
cd /root/brief-bot
npm install
pm2 start bot.js --name brief-bot
```

---

## Шаг 8 — Настроить автозапуск

Чтобы бот запускался автоматически после перезагрузки сервера:

```bash
pm2 startup
pm2 save
```

---

## Шаг 9 — Проверить что бот работает

```bash
pm2 status
pm2 logs brief-bot
```

Должно быть видно: `Бот запущен` и статус `online`.

---

## Шаг 10 — Протестировать

1. Открыть Telegram → найти `@Lendingferst_bot`
2. Отправить `/start`
3. Нажать кнопку «Заполнить бриф»
4. Пройти все шаги и нажать «Отправить»
5. Убедиться что сообщение с брифом пришло в личку Виктории

---

## Полезные команды PM2

```bash
pm2 status              # статус бота
pm2 logs brief-bot      # логи в реальном времени
pm2 restart brief-bot   # перезапустить бота
pm2 stop brief-bot      # остановить бота
```

---

## Если что-то пошло не так

1. `pm2 logs brief-bot` — смотреть ошибки
2. Проверить что `.env` заполнен правильно
3. Проверить что `BOT_TOKEN` не просрочен (отозван)
