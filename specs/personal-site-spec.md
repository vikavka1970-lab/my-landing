## Визуальный стиль (рекомендация)

**Направление: «Clean Dark Professional»**

Выбрано исходя из:
- B2B аудитория → нужно профессиональное, не игривое
- AI/tech услуги → тёмная тема усиливает tech-credibility
- Нет ярких кейсов → стиль сам по себе должен говорить о качестве

**Цветовая палитра:**
```
Фон:              #0F172A  (глубокий тёмно-синий)
Поверхность:      #1E293B  (карточки, блоки)
Акцент основной:  #6366F1  (indigo — кнопки, ссылки, hover)
Акцент вторичный: #06B6D4  (cyan — иконки, подчёркивания)
Текст основной:   #F1F5F9  (почти белый)
Текст muted:      #94A3B8  (серый — второстепенный текст)
```

**Типографика:**
```
Заголовки: Space Grotesk, 700-800 weight
Текст:     Inter, 400-500 weight
(обе бесплатно на Google Fonts)
```

**Визуальные детали:**
- Тонкий gradient mesh на hero-фоне (фиолетово-синий)
- Glassmorphism на карточках услуг (backdrop-filter: blur)
- Плавные fade-in анимации при скролле
- Hover-эффект на кнопках — лёгкое свечение

## Технический стек (рекомендация для вайбкодинга)

```
Фреймворк:    Next.js 14 (App Router)
Стили:        Tailwind CSS
Анимации:     Framer Motion (простые fade-in, hover)
Шрифты:       Google Fonts (Space Grotesk + Inter)
Деплой:       Vercel (бесплатно, в 1 клик)
Форма:        Formspree или Resend (бесплатные тарифы)
```

**Почему этот стек для вайбкодинга:**
- Next.js + Tailwind = самая популярная комбинация, AI генерирует отлично
- Framer Motion = простые анимации без сложного кода
- Vercel = деплой в 1 клик прямо из GitHub
