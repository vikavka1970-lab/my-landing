# project.md — Личный сайт Виктории Полтавченко

## Что это за проект

Личный сайт Виктории Полтавченко — веб-разработчика и специалиста по AI-автоматизации. Сайт решает две задачи одновременно: строит личный бренд и привлекает B2B-клиентов через форму обратной связи. Несколько HTML-страниц, никаких фреймворков — чистый HTML/CSS/JavaScript, хостинг на GitHub Pages.

**Живой сайт:** https://vikavka1970-lab.github.io/my-landing/

### Технические детали
- `index.html` — главный лендинг (~1900 строк), `pricing.html` — страница тарифов
- Стили и JS — внутри каждого HTML-файла, общего CSS нет
- Изображения хранятся в репозитории в папке `images/` — резервная копия есть в git
- Кастомный курсор (dot + ring)
- IntersectionObserver: fade-up анимации при скролле
- 3D-tilt на карточках услуг и портфолио
- Магнитные кнопки (hover-притяжение)
- Typewriter-эффект в Hero
- Прогресс-бар скролла вверху страницы
- Адаптивная вёрстка (мобайл, планшет, десктоп)
- Favicon SVG с инициалами «ВП»
- OG-теги для шеринга в соцсетях

## Стиль дизайна

**Направление:** Dark Futurist — тёмная тема с технологичной эстетикой

**Цветовая палитра:**

| Переменная | Цвет | Применение |
|---|---|---|
| `--bg` | `#0F172A` | фон страницы |
| `--surface` | `#1E293B` | карточки, блоки |
| `--border` | `#334155` | границы |
| `--accent` | `#6366F1` | кнопки, ссылки (indigo) |
| `--cyan` | `#06B6D4` | иконки, акценты |
| `--text` | `#F1F5F9` | основной текст |
| `--muted` | `#94A3B8` | второстепенный текст |

**Шрифты:**
- `Syne` (700–800) — заголовки, навигация, кнопки
- `DM Sans` (400–500) — основной текст
- `Syne Mono` — метки, теги стека

**Визуальные элементы:**
- Aurora gradient mesh на Hero (медленно двигающиеся орбы)
- Glassmorphism на карточках (`backdrop-filter: blur`)
- Grain texture overlay на всём сайте (тонкий шум, opacity 3.5%)
- Gradient underline под заголовками секций
- Зернистость + глубина = «живой», не плоский экран

## Принципы, которые нельзя нарушать

- **Никаких фейковых отзывов** — только реальные клиенты с именем и должностью
- **Никаких выдуманных цифр** — счётчики проектов убраны намеренно
- **Честность** — сайт строит доверие, а не имитирует его
