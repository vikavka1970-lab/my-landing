# CLAUDE.md — Telegram Mini App «Бриф-бот»

> Интерактивный бриф прямо в Telegram: клиент проходит 5 шагов,
> нажимает «Отправить» — Виктория получает оформленную заявку.

---

## Файловая структура

```
tg-app/
├── index.html   ← Весь Mini App: HTML + CSS + JS в одном файле
└── CLAUDE.md    ← Этот файл
```

**Всё находится в `index.html`:**
- Стили — внутри `<style>` в `<head>`
- Разметка экранов — в `<body>` внутри `#app`
- Логика — внутри `<script>` в конце `<body>`

---

## Экраны и их назначение

| ID экрана | Назначение | Переход |
|---|---|---|
| `#screen-0` | Приветствие: аватар, заголовок, кнопка «Начать» | → 1 (кнопка) |
| `#screen-1` | Тип проекта: Сайт / Приложение / AI | → 2 (автопереход при выборе) |
| `#screen-2` | Срок: Срочно / Стандартно / Не горит | → 3 (автопереход) |
| `#screen-3` | Бюджет: 4 варианта | → 4 (автопереход) |
| `#screen-4` | Приоритеты: 4 варианта, множественный выбор | → 5 (кнопка «Далее») |
| `#screen-5` | Контакт: имя/ник из Telegram, поля телефон и детали | → 6 (кнопка «Отправить») |
| `#screen-6` | Успех: галочка, кнопка «Написать напрямую» | закрытие |

## Навигация между экранами

**Функция `go(to, dir)`** — единственная точка перехода.

```javascript
go(1, 'forward')  // перейти вперёд на экран 1
go(3, 'back')     // вернуться назад на экран 3
```

**Анимация:**
- `'forward'` — новый экран въезжает справа (translateX 105% → 0), старый уходит влево (-35%)
- `'back'`    — новый экран въезжает слева (-105% → 0), старый уходит вправо (105%)
- Длительность: 250ms, `ease`

**Кнопки Telegram:**
- `tg.MainButton` — нижняя нативная кнопка (текст меняется по шагам)
- `tg.BackButton` — стрелка назад в хедере (показывается на экранах 1–6)

**Fallback-кнопки** (`.fb-btn`) — видны только при тестировании в браузере без Telegram.

## Где менять данные

### Контактная ссылка (экран 6)
```html
<!-- tg-app/index.html, экран #screen-6 -->
<a href="https://t.me/ЗДЕСЬ_НИК" ...>
```
И в JS:
```javascript
tg.openTelegramLink('https://t.me/ЗДЕСЬ_НИК');
```

### Варианты карточек
Каждая карточка — `<div class="card" data-value="ТЕКСТ_В_БОТ" onclick="pickOne(this, N)">`.
Атрибут `data-value` — это строка, которая попадёт в бриф.

### Аватар Виктории (экран 0)
Сейчас: инициалы «ВП» через CSS-градиент.
Когда будет фото — заменить блок:
```html
<!-- Было: -->
<div class="welcome-avatar">ВП</div>

<!-- Станет: -->
<img src="victoria.png" alt="Виктория Полтавченко" class="welcome-avatar" style="object-fit:cover;">
```

### Статус доступности (экран 0)
```html
<div class="status-badge">
  <span class="status-dot"></span>
  Открыта для новых проектов  ← менять здесь
</div>
```

## Что получает Виктория в боте

При нажатии «Отправить» вызывается `tg.sendData(JSON)`.
Бот получает событие `web_app_data` с таким объектом:

```json
{
  "name": "Максим Иванов",
  "username": "maxim_ivanov",
  "type": "Сайт / лендинг",
  "timeline": "Стандартно (1–2 месяца)",
  "budget": "30 000 – 80 000 ₽",
  "priorities": "Дизайн и качество, Прозрачность и общение",
  "contact": "+7 999 000 00 00",
  "details": "нужен лендинг для кофейни, есть брендбук"
}
```

**Бот (ещё не создан)** должен:
1. Слушать `bot.on('web_app_data', ctx => { ... })`
2. Распарсить JSON из `ctx.webAppData.data`
3. Отправить форматированное сообщение в личку Виктории

## Тёмная и светлая тема

Цвета берутся из переменных Telegram SDK автоматически:
```css
--bg:      var(--tg-theme-bg-color, #ffffff)
--surface: var(--tg-theme-secondary-bg-color, #f4f4f5)
--text:    var(--tg-theme-text-color, #000000)
--accent:  var(--tg-theme-button-color, #2aabee)
```

Переключение темы — на стороне Telegram, Mini App подстраивается сам.

## Тестирование в браузере

Открыть `tg-app/index.html` в браузере — работает как обычная страница.
При этом:
- Показываются HTML fallback-кнопки (`.fb-btn`) вместо Telegram MainButton
- `tg.sendData()` не вызывается (ловим ошибку в try/catch)
- Данные пользователя недоступны → показывается «Гость»

## Что нужно создать для полноценной работы

| Шаг | Что делать |
|---|---|
| 1 | Создать бота через `@BotFather`, получить токен |
| 2 | Задеплоить `tg-app/index.html` (Vercel, GitHub Pages, любой хостинг с HTTPS) |
| 3 | В BotFather настроить Menu Button → URL задеплоенного файла |
| 4 | Написать бот на Node.js (Telegraf): слушать `web_app_data`, отправлять сообщение |
| 5 | Добавить кнопку на лендинге: ссылка `https://t.me/ИМЯ_БОТА?start=brief` |

## Технические ограничения v1

- Загрузка файлов — не поддерживается (отправляется потом в чат напрямую)
- Date picker — намеренно заменён кнопками «срочно / стандартно / не горит»
- Редактирование предыдущих ответов — только через BackButton
